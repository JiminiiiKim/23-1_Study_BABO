### 문제 풀이 핵심 아이디어 :
        DFS(깊이 우선 탐색) : 그래프에서 깊은 부분을 우선적으로 탐색하는 알고리즘

        그래프의 구조
        - 노드(정점이라고도 함.), 간선
        -> 두 노드가 간선으로 연결되어 있다. == 두 노드가 인접하다.
        
        표현하는 방식
        1. 인접 행렬 : 2차원 배열로 그래프의 연결 관계를 표현하는 방식
        - 2차원 배열에 각 노드가 연결된 형태를 기록하는 방식
        - 연결되지 않은 노드끼리는 무한 비용(INF)로 작성함.
        2. 인접 리스트 : 리스트로 그래프의 연결 관계를 표현하는 방식

### 코드 :
```python
# 반복적으로 구현한 n!
INF = 9999999

graph = [
    [0, 7, 5],
    [7, 0, INF],
    [5, INF, 0]
]
    
print(graph)
```
