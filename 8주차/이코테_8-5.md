### 문제 풀이 핵심 아이디어 :
    다이나믹 프로그래밍 (보텀업) 방식으로 문제 해결함.

### 코드의 시간복잡도와 그 이유 :
    O(x) : 반복문을 돌며 각 x 값에 해당하는 최솟값을 구함.

### 코드 :
```python
x = int(input())

d = [0] * 30001

for i in range(2, x+1) :
    d[i] = d[i-1] + 1
    if i % 2 == 0 :
        d[i] = min(d[i], d[i // 2] + 1)
    if i % 3 == 0 :
        d[i] = min(d[i], d[i // 3] + 1)
    if i % 5 == 0 :
        d[i] = min(d[i], d[i // 5] + 1)

print(d[x])
```
